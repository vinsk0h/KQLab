| where ResultType == "50126"
| summarize
    FailedAttempts = count(),
    TargetedUsers  = dcount(UserPrincipalName),
    Locations      = make_set(Location)
    by IPAddress
| where TargetedUsers > 5
      and FailedAttempts > 20
| order by FailedAttempts desc

Self-Host

Deploy in minutes

Docker Compose or bare Node.js. SQLite included — no external database required.

🐳

Docker Compose

Recommended 
# Clone and configure
$ git clone https://github.com/vinsk0h/KQLab
$ cd KQLab
$ cp .env.example .env

# Generate secrets (run twice)
$ npm run keygen

# Start production instance
$ docker compose up -d
# → http://localhost:3000
⚡

Node.js Direct

# Requirements: Node.js ≥ 18
$ git clone https://github.com/vinsk0h/KQLab
$ cd KQLab
$ npm install
$ cp .env.example .env

# Fill .env then start
$ npm start
# → http://localhost:3000
